# Magnet
It just clicks. Euphorically ensure security of your LLM and Agentic app.

# Usage
## Build
```make docker-build``` 

## Run
Run the application inside the docker container and expose it on port `8000`
```make docker-run```

## Dnd storybook
To visualize and play around in the dnd storybook run
```make docker-it```
